# Pizza Sales Dashboard - Power BI

## Project Overview
This project is an interactive **Power BI dashboard** built to analyze pizza sales performance. It provides key business insights through KPIs, sales trends, and category analysis to help stakeholders monitor performance and make data-driven decisions.

## Objectives

- Analyze overall sales performance.
- Monitor key business KPIs.
- Identify best-performing pizza categories and sizes.
- Analyze daily and monthly order trends.
- Provide an interactive dashboard for business users.



## Key Performance Indicators (KPIs)

- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value


## Dashboard Features

- Sales by Pizza Category
- Sales by Pizza Size
- Daily Trend of Total Orders
- Monthly Trend of Total Orders
- Total Pizzas Sold by Category
- Best and Worst selling pizzas
- Interactive navigation between report pages
